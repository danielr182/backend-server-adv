# BackEnd-Server

Este es el código necesario para establecer el backend conectado a MongoDB usando Mongoose.

Para ejecutarlo, es necesario reconstruir los módulos de node usando el comando

````
npm install
````

Para subir el servidor:

````
npm run dev
````
